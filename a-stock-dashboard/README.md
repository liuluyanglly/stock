# 📈 A股全自动复盘看板 v5

基于 Flask + akshare + SQLite 的 A 股每日复盘看板，单文件架构，开箱即用。

## 功能一览

| 模块 | 说明 |
|------|------|
| 🏠 首页 | 大盘指数、市场情绪、涨停/跌停/炸板概览 |
| 🔴 涨停股 | 涨停一览（连板、首封时间、封板资金、涨停原因） |
| 🟢 跌停股 | 跌停一览（连续跌停、封单资金） |
| 🟡 炸板股 | 炸板一览（炸板次数、振幅） |
| 🔥 热门股票 | 涨停热门、跌停热门、炸板热门、主线板块 |
| 💰 主力净流入 | 主力资金 TOP 股票 |
| 📊 昨日涨停 | 昨日涨停股今日表现 |
| 📈 新高票 | 创近一年新高股票 |
| 📰 财联社新闻 | 实时财经快讯 + 情绪分析 |
| 📈 趋势票 | 上升趋势股票筛选 |
| 🔄 K线反包 | 阳包阴形态识别 |
| 📊 板块统计 | 板块涨跌分布、资金流向 |
| 💰 板块资金 | 板块资金流向（卡片/分栏/列表三视图） |
| 👑 板块龙头 | 行业龙头 + 跟风股分析 |
| 🏷️ 分组概念 | 自定义概念分组管理（拖拽排序） |
| 💼 持仓管理 | 个人持仓记录 + 实时盈亏 |
| 🎯 想买列表 | 自选股 + 买入理由 + 目标价 |
| 🤖 AI预测 | MiMo-V2.5-Pro 智能复盘分析（9项分析） |
| 📋 预测历史 | AI 预测历史记录 |
| 💊 重组公告 | 资产重组公告追踪 |
| 💎 基金推荐 | 基金推荐列表 |
| ⚡ 异动预测 | 盘口异动预测 |
| 🛡️ 监控室 | 实时盘口异动 + 风险提示 + ST 股票 |

## 技术栈

- **后端**: Python Flask（单文件，~6300 行）
- **数据源**: akshare（A股数据接口）
- **数据库**: SQLite（daily_snapshots 表）
- **AI**: MiMo-V2.5-Pro（Anthropic 兼容 API）
- **前端**: 原生 HTML/CSS/JS（内嵌在 Python 字符串中）
- **UI**: 深色主题，紫色主色调，Lucide 图标

## 快速启动

```bash
# 安装依赖
pip install flask akshare requests

# 设置 API Key（AI 预测功能需要）
export MIMO_API_KEY="your-api-key-here"

# 启动（需要 sudo 因为监听 80 端口）
sudo -E python3 stock_app.py
```

访问 `http://localhost` 即可。

## 数据流

```
akshare API → Python 后端 → SQLite 存储 → JSON API → 前端渲染
```

## 目录结构

```
a-stock-dashboard/
├── stock_app.py    # 单文件 Flask 应用（后端 + 前端全量代码）
├── .gitignore
└── README.md       # 本文件
```

## 注意事项

- 需要 **sudo 权限**（监听 80 端口）
- akshare 数据获取可能受网络影响，建议在 A 股交易时间后运行
- AI 预测功能需要配置 `MIMO_API_KEY` 环境变量
- 持仓/想买/概念分组数据存储在浏览器 localStorage
